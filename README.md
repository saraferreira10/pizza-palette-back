## 🔧 Instalação

Clone o repositório:
```
git clone https://github.com/saraferreira10/pizza-palette-back
```

Acesse o diretório principal:
```
 cd pizza-palette-back
```

Inicie o projeto:
```
mvn spring-boot:run
```
